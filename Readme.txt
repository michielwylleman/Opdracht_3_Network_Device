Documentatie
---------------------------------------------------------------------------------------
Login in https://developer.cisco.com/site/sandbox/ en kies reseveer de sandbox van cisco IOS XE	16.9.3
Log dan in je gereseveerde VPN met de volgende gegevens (zie je emails)
Je kan zowel SSH inloggen als ook via RESTCONF en NETCONF

Mogelijk problemen
--------------------------------------------------------------------------------------
Als je deze foutmelding krijgt: "ImportError: No module named requests"; zal je moeten opnieuw opstarten of request opnieuw moeten herinstalleren.
